Aprendiendo HTML

Gracias a estas extensiones (o como las quieran llamar) hemos hecho posible esta aplicación para que puedan aprender HTML5:

La Comunidad


Gracias a la comunidad, por compartir sus codigos, estilos y hacerlos accesibles a todos nosotros. No podría contar todas las webs que he consultado y he tomado trozos (muchos trozos de codigo) que han acabado formando esta aplicación. ¡GRACIAS!

Sukafe

Es un grupo de desarrolladores ubicados en Sevilla con el objetivo de implementar los lenguajes web en nuestro día a día.

Brackets

Brackets es un IDE desarrollado por Adobe con HTML, CSS y JavaScript.
Permite la instalación de extensiones y también posee una herramienta llamada Desarrollo en Vivo que hace más facil ver los cambios en el diseño.

Mozilla Firefox

Durante el desarrollo de esta aplicación hemos utilizado Mozilla Firefox
No hay mucho más que contar que ya no se sepa ;)

Firefox OS Simulator

Firefox OS Simulator es un complemento de Mozilla Firefox donde poder probar las aplicaciones que se van desarrollando para Firefox OS

Chrome & Chrome Canary


Hemos utilizado tanto Chrome como Chrome Canary realizar pruebas con sus herramientas de diseño

Photoshop

Photoshop (PS para los amigos) es un editor de gráficos desarrollado por Adobe, con el cual podemos editar imagenes con muchísimas posibilidades

Spotify & Youtube


Nada de esto hubiera sido posible sin la música que escuchamos mediante Spotify y Youtube :)

BackStrech

Es un simple plugin de jQuery que te permite agregar imagenes de fondo de una forma dinámica redimensionada para que sea capaz de lucir en cualquier página o elemento.

intro.js

Aunque ya no se use a partir de la version 0.8, gracias a intro.js pudimos realizar una pequeña guia en nuestra app para ayudar a algunos usuarios a entender como funciona.

Raphaël Icon-Set

Gracias a Raphaël Icon-Set hemos podido disponer de una amplia biblioteca de iconos para la app (Sustituido por Font Awesame).

basic.css

basic.css es un archivo de estilos predefinidos por clases creado por Sukafe que hace más fácil crear diseños.

Building Blocks

Building Blocks nos ha proporcionado guias y estilos para hacer la app al más puro estilo Firefox. Aprendiendo HTML no está hecha sobre Building Blocks pero sí hemos utilizado su menú lateral que funciona genial. (Sustituido por Slicknav)

Registro y Login


El diseño del Registro (Que originalmente era solo un Login) lo realizó un desarrollador con Lungo, el cual compartió su codigo y pudimos conectar aquel diseño (que solo era fachada, nada funcionaba) a nuestra base de datos y utilizar como Login y Registro.

Font Awesome

Font Awesome es una librería increible y muy fácil de usar con muchísimos iconos :)

Slicknav

Increible addon para poder crear un nuevo menú con un bonito desiño y fácil de modificar.
