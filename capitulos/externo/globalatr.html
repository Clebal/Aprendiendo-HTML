Atributo Descripción

accesskey

 Especifica una tecla de acceso directo para activar un elemento.

class

 Especifica uno o más nombres de class para un elemento (se refiere a class en una hoja de estilo)

contenteditable

 Especifíca si un elemento es editable (por el usuario) o no.

contextmenu

 Especifica un menú contextual de un elemento. Aparecerá el menú contextual cuando un usuario hace clic derecho en el elemento.

data-*

 Se utiliza para almacenar datos personalizados ocultos de la página o aplicación (Pueden aparecer al darle clic)

dir

 Especifica la dirección (izq o derecha) del texto en un elemento.

draggable

 Especifica si un elemento se pueda arrastrar o no.

dropzone

 Especifica si los datos arrastrados seran copiados, movidos, o vinculados, cuando se deja caer tal elemento

hidden

 Especifica todavia no es todavía o ya no es relevante

id

 Especifica una unica id para el elemento (Atributo de estilo)

lang

 Especifica el lenguaje del contenido del elemento

spellcheck

 Especifica si el elemento debe tener en cuenta (y comprobar) su ortografía y gramática, o no

style

 Especifica el estilo de una etiqueta en la misma etiqueta, no en un archivo .css

tabindex

 Especifica el orden al seleccionar las etiquetas mediante el tabulador

title

 Especifica una información extra de un elemento (Cuando se deja el cursor encima del elemento aparece la información extra)

translate

 Especifica si el contenido de un elemento debe ser traducido o no
