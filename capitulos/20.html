Antes hemos tardado muy poco en ver la etiqueta <wbr> pero ahora vamos a hablar de muchas etiquetas (Son bastantes jaja) que sirven básicamente para darle estilo a un texto, ya sea ponerlo en cursiva, negrita o subrayarlo, vamos por las dos primeras.

<b></b> y <strong></strong>

Las dos etiquetas sirven para poner en negritra el texto que esté dentro de la etiqueta de inicio y de cierre.

<b>Sukafe :)</b>

La etiqueta <strong> sin embargo ayuda a que cuando alguien busque por Google, encuentre nuestra página primero. Por ejemplo:

Si tengo una página que vende móviles y pongo en la etiqueta <strong> palabras como: Bienvenido, contacto, envio. Pues alguien cuando busque en Google una página de venta de moviles, no aparecerá nuestra página primero, pero si ponemos: vender, moviles, calidad, precio bajo. Entonces sí :) Google tiene en cuenta las palabras (una o más) que esten en la etiqueta <strong>.

<em> <i> <dfn> <var> <cite>

No os voy a engañar, las cinco son iguales a la hora de ser mostradas al usuario que visita nuestra web, basicamente, todas las palabras que esten entre esas etiquetas (<i>Sukafe</i> Por ejemplo) tendrán este código CSS:

font-style: italic;

La diferencia reside basicamente en el valor que tienen, al igual que <strong> y <b>, hacen lo mismo pero no valen igual ya que el navegador lo interpretará de una forma.

<em> Se usa para darle enfasis a un texto poniendolo en cursiva.

<i> se puede utilizar para indicar un término técnico, una frase de otra lengua, un pensamiento, o el nombre de algo, etc.

<dfn> indica una definición.

<cite> se utiliza para escribir la cita.

<var> contiene texto que supone ser la instancia de una variable o el argumento de un programa. El contenido de este elemento es usualmente mostrado por los agentes de usuario con un estilo de fuente itálica. (Esta etiqueta es bien extraña eh).

<kbd> <samp> <code>

De nuevo más de lo mismo... ¡Visualmente son lo mismo! ;) Estas etiquetas utilizan este código CSS:

font-family: monospace;

<samp> se utiliza para un código que sería el producido por un programa del ordenador.

<kbd> se utiliza para un código introducido por el teclado.

<code> se utiliza para escribir las lineas de código ya sea HTML, C, el que sea :)

<mark>

La etiqueta <mark> se utiliza para destacar un texto poniendolo como subrayado con fluorescente. Muchos de los que lo utilizamos para estudiar lo conocerán muy bien ;) Este es su código CSS:

background-color: yellow;
color: black;
