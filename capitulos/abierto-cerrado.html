Abierto o Cerrado

Una de las cuestiones que más se dan entre los que se inician en HTML es cuales etiquetas se dejan abiertas y cuales necesitas de un cierre, osea ser cuales necesitan una etiqueta que termine </etiqueta> y cuales no.

La explicación es muy sencilla, consiste en lo siguiente:

Una etiqueta que no necesita etiqueta de cierre (</etiqueta>) quiere decir que mediante los atributos se completa y no necesita un contenido. Lo veremos mejor con un ejemplo:

<img src="img/cabeza.png" alt="">

La etiqueta <img> no se cierra debido a que mediante el atributo src=""(Que sirve para encontrar la imagen) y el atributo alt="" (Que sirve como texto alternativo en caso de que no se muestre la imagen) se completa sin necesidad de que exista un </img> ya que no tiene sentido poner un texto dentro de una imagen.

Después existen otras etiquetas que si necesitan ser cerradas como <p>, esta etiqueta se utiliza para escribir texto, lo que hace obligatorio decir: de aquí hasta aquí es texto y debe ser mostrado como tal. En la etiqueta <img> que pusimos antes de ejemplo, no necesita que se le diga de donde hasta donde pues en la misma etiqueta se le dice lo que necesita para funcionar.
