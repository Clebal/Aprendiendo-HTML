Otras etiquetas

Pues hasta aquí llegó el curso, habrá dos páginas más sobre como enviar nuestra aplicación a Firefox pero esta será la última página de etiquetas. Pero... ¿Por qué otras etiquetas? ¿Por qué eso de "otras"? Bien sencillo de explicar, las etiquetas que veremos a continuación son una serie de etiquetas que HTML usa para interactuar con otros lenguajes (como PHP o JavaScript), estos lenguajes no los vamos a enseñar aquí, quien sabe una futura aplicación ;) Por ello, vamos a aprender esas etiquetas y cuando vayais avanzando con nuevos lenguajes sabréis usarlas, vamos a ello!

<embed>

La etiqueta <embed> se utiliza para mostrar archivos que no son HTML, en muchos casos archivos de extensión .swf :)

<button>



Sí! Eso es un botón con un pequeño código JavaScript, este botón se hace con el siguiente código:


<button> Clicame! </button>

Esta etiqueta se suele utilizar para ejecutar codigos JavaScript

<form>, <input> y <textarea>

Estás etiqueta se usan a la hora de crear formularios.

<form> engloba un formulario, el cual por lo general va conectado a un código PHP.

<input> tiene un atributo obligatorio que es type el cual determina que tipo de input será. Input sirve para recoger datos que da el usuario, por ejemplo: type="text" creará un recuadro en el que introducir un pequeño texto como el nombre de usuario. Hay muchos tipos más.

<textarea> es una etiqueta que se suele utilizar dentro de form. Funciona igual que input type="text" pues define un recuadro pero muy grande, para introducir largos textos, se podria comparar con los recuadros que hay para escribir entradas en un blog :)

Terminamos aquí la mayoría de las etiquetas de HTML5, espero que les haya gustado :) Aquí termina el curso pero seguramente en proximas actualizaciones vayamos añadiendo más cosas sobre aplicaciones Firefox OS! Gracias por todo!
