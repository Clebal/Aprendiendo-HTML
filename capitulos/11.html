CSS3 & HTML5

CSS es un lenguaje de hojas de estilos que se usa para describir como se presentaran las etiquetas que escribamos, lo que quiere decir que le dará estilo a nuestra web.

La diferencia es bastante grande entre un página con HTML5 y CSS3, con otra que no tiene CSS3.

A nuestra web podemos añadir estilos de tres formas distintas:

<link>


La etiqueta <link> ya la dimos en la página anterior por lo que aquí solo haremos un pequeño repaso.

Recordemos que la etiquta <link> se utiliza para enlazar un archivo externo con nuestro documento, en este caso enlazaremos el archivo .css con el .html. Esta primera opción para añadir estilos a HTML es mediante un archivo externo, esta es para mi la mejor opción convinada con la tercera.

<style>


Esta etiqueta (Que por supuesto se cierra) define un documento .css en el mismo documento .html. Esto es bueno salvo por el pequeño detalle de que si vamos a escribir un .css muy largo y un .html muy largo, si todo lo unimos en la misma página se hará enorme y más dificil a la hora de buscar fallos en el codigo. Atributo style=""


Con el atributo style="", que pondremos en cada etiqueta, podemos añadir estilos directamente en la etiqueta sin necesidad de enlazar los elementos de HTML con los estilos del CSS con ID y Class
