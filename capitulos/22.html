Antes de pasar a otros apartados más gordos que van a finalizar este curso, os quiero enseñar una etiqueta bastante interesante que les ayudará...

<base>

Cuando tienes una web con muchas imagenes (por ejemplo, puede ser un enlace también) y estas están en la misma carpeta podemos utilizar este truco para no comernos la cabeza ;)

<base> haciendo uso del atributo href="" define una base para todas las url que aparezcan, vamos con un ejemplo, peeeero antes espera que te diga una cosa, esta etiqueta no se pone dentro de la etiqueta <body> sino dentro de la etiqueta <head>, ahora sí, vamos con el ejemplo:

<head>
    <base href="http://www.example.com/img/">
</head>

<body>
    <img href="example.png" alt="Example">
</body>

Recordad que la etiqueta img debe llevar siempre siempre el atributo alt="" ;)

En el ejemplo de arriba podeis ver que dentro de la cabeza de nuestra página creamos una base para todos los atributos href y src. Así en la etiqueta <img> cuando ponemos el atributo href indicando donde esta la imagen, no hace falta nada más que introducir el nombre de la foto y la extensión del archivo.

Pero claro, ¿qué ocurre si tengo un enlace que me lleva a otra página? Lo que debemos hacer en ese caso es sencillamente indicar el http://www.sukafe.com, toda la URL completa, incluido http:// y el www, con esto el navegador ignorará en ese caso la etiqueta <base>
