Todavia no vamos a entrar en las hojas de estilos en cascada (CSS por sus siglas en Ingles, Cascading Style Sheets), se que tal vez tienes ganas pero tendras que esperar a la siguiente página para empezar a aprender, lo siento ;)

Ahora veremos como enlazar un documento externo con nuestro archivo HTML (El index.html es este caso). Podemos enlazar muchas cosas mediante la etiqueta <link>, pero nosotros estamos viendo lo básico para construir una web, por lo que aprenderemos a enlazar un archivo .css (Archivo de estilo) con nuestro archivo .html (Claramente los llamamos .css y .html por la extensión del archivo que se genera)

Las estructura general de la etiqueta <link> es así:

<link rel="stylesheet" type="text/css" href="css/style.css">


Supongamos que hemos creado la carpeta "Sukafe HTML", dentro tenemos dos carpetas "css" e "img" (Esta es la estructura básica de una web), en la carpeta "css" se encontraran los estilos y la "img" las imagenes (Ya veremos más adelante como utilizar imagenes).

Vemos en el ejemplo que para seleccionar el archivo style.css que está dentro de la carpeta "css" debemos escribir css + / + nombrearchivo.css (en este caso style.css). Es sencillo ¿no?

Si tuvieramos el archivo index.html dentro de una carpeta hariamos lo siguiente...

.. + / + css + / nombrearchivo.css

(Espero que no haya que aclarar que los + no se ponen ;) Los dos puntos quiere decir que vaya una carpeta atras, si el archivo index.html está en la carpeta "hola" pues ira para atras situandose en la carpeta principal, luego entrará en la carpeta llamada "css" y buscará el archivo nombrearchivo.css

Vemos tres atributos, el atributo rel="", el type="" y href="".

El atributo rel="" y el type="" indican casi lo mismo, que el archivo que estamos enlazando es de estilos, pero se usan para que nuestra web sea compatible con antiguas versiones de algunos navegadores (Internet Explorer) que aun siguen siendo utilizadas. El atributo rel="" es obligatorio Sí o Sí, podemos ver los diferentes tipos de valores que puede tomar rel="" en la página de W3Schools. Como dije antes aquí veremos lo básico y eso es algo más avanzado.

Una vez que hemos enlazado el archivo .css a nuestro index.html podemos empezar, ahora sí, a aprender CSS3 para darle estilo a HTML5.
