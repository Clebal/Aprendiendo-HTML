Todos sabemos que cuando iniciamos una web aparece en la pestaña del navegador el titulo de la web. También esta web aparece en las busquedas de Google y tiene una descripción, otra cosa es que la web se ve bien, tiene una estetica excelente. Todo ello se reune en una etiqueta llamada:

<head>

La etiqueta <head> se compone de varias etiquetas con las que obtendremos las cosas mencionadas arriba. <head> es, al igual que <html> o <body>, un contenedor que reune muchas etiquetas.

Cuando el navegador está leyendo el código y se encuentra con la etiqueta <head> que...

¿Se cierra o no, la etiqueta <head>?


 A. Sí
 B. No
 C. Sí pero con la etiqueta <//head>

Esta etiqueta no tiene un atributo especifico, sin embargo existe una serie de atributos que sirven para todas las etiquetas llamados:

Atributos Globales

Si utilizamos una versión antes de HTML5, los atributos globales no serán para todas las etiquetas.

Pero como nosotros vamos a trabajar con HTML5 tendremos en cuantas que los atributos globales sirven para todas las etiquetas que escribamos

Cabe destacar que el que sirvan para todas las etiquetas no quiere decir que tengan sentido o utilidad en todas.
